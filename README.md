\# backend-enterprise-onboarding



A 14-day enterprise backend engineering onboarding project.



\## Stack

\- NestJS

\- TypeScript

\- Prisma

\- PostgreSQL

\- Redis

\- Docker



\## Purpose

This repository follows the company's engineering standards track,

covering git workflow, architecture principles, testing, CI/CD,

and production-grade backend development.



\## Setup

1\. Install dependencies: `pnpm install`

2\. Copy environment variables: `cp .env.example .env`

3\. Start the database: `docker compose up -d`

4\. Run the app: `pnpm start:dev`
